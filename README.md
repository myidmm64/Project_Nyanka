## 🍰 서론

- *게임 제목* : 냥카츠키
- *게임 장르* : 3D 턴제 RPG
- *플랫폼* : PC, 안드로이드 모바일
- *제작 인원* : 2 인
- *사용 엔진과 언어* : Unity, C#
- *게임 설명* : 군단장들의 소행으로 뿔뿔이 흩어지게 된 냥카츠키. 매 턴마다 최고의 선택을 하여 모든 군단장들을 쓰러트려라! 프레스턴과 각성 시스템을 사용한 턴제 게임!
